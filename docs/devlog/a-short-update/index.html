A short update

October 2009

My second iPhone game is progressing well but at a slow pace (as expected). I’m really stretching my digital art skills on this one, investing a huge amount of time creating the artwork and interface. These both need to be spot on if this game is to stand out in the App store.

I’m also spending some time exploring the gameplay mechanics. The core idea is simple and I want to enhance it with appropriate intricacies to ensure that it holds peoples attention. I’ve already got an idea for a “multiplayer” mode of sorts which I hope will add to the longevity.

As it stands, i’m currently working towards a Christmas release.
