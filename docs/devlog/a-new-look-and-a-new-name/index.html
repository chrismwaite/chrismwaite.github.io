A new look and a new name

June 2009

Just a short post this time. I’ve made some quite big changes in the last week. Firstly and most importantly, the Icarus theme has now taken a backseat (although I may still use it for some inspiration in the game). Along with the dropped theme is a new working title, “Sunlight”.

I’ve spruced up the graphics somewhat although they are still likely to change considerably before the game is released. Here is the first screenshot of the game under its new moniker “Sunlight”.
