...And so it begins!

March 2009

After a month or so of conceptual scribbles and learning the basics of iPhone programming, i’ve finally begun to develop my first game. I’ve started by testing a few interface designs alongside coding the basic framework. The end is a long way off but it feels good to have finally started.

For anyone interested, I’m reading “Beginning iPhone Development” by Dave Mark and Jeff LaMarche in order to get up to speed on the basics.
