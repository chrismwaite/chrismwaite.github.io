Talking to the game world

April 2009

Progress has been pretty slow on the game (mainly due to other responsibilities).

I have written a basic php page that talks to my database (which will hold the game world) and outputs the content in XML. Simple stuff, but it’s an important step in the games development. I’m currently figuring out the XML parsing on the iPhone so that I can read the XML and recreate the game world on the iPhone.

On a side note, the feature set is growing and UGC (user generated content) is becoming even more central to the game. To such an extent that I wonder if people will get it! Time will tell…
